Caveats

  • There is currently no code to add animated png support for non-supporting browsers, so to see animated emotes actually animate, you should either use Firefox or Chrome with the APNG extension installed.
  • There is currently nothing to disable NSFW emotes, so they show up here as well
  • The coloring modifiers (-i and -invert) don't currently render into the rendered emote, just into the emote text
  • There is currently no support for the lines of text you can add to some emotes like the 'sign' emotes
  • There is only support for one emote currently, but having 2 or 3 emotes (especially trying to combine them) will be added
